### 3.1 荷载计算测试

#### 测试1：永久作用（自重）
```
输入: DN1000管道，跨径20m
截面积: 0.031730 m³/m
钢材密度: 7850 kg/m³
自重放大系数: 1.2

理论值: 2.4435 × 1.2 = 2.9322 kN/m
计算值: 2.9322 kN/m
误差: 0%
结论: ✅ 通过
```

#### 测试2：工况组合逻辑
```
工况1（基本组合）:
  - 竖向永久荷载: 自重+防腐+附加+水重
  - 竖向可变荷载: 0 (内水压/真空压不产生重力)
  - 结论: ✅ 物理逻辑正确

工况2（施工检修）:
  - 竖向可变荷载: 施工荷载
  - 结论: ✅ 物理逻辑正确
```

### 3.2 应力计算测试

#### 测试3：弯曲应力
```
输入: 线荷载 15.09 kN/m, 跨长20000mm
理论弯矩: qL²/8 = 15.09 × 20000² / 8 = 7.54×10⁸ N·mm
截面模量: 7.94×10⁶ mm³
理论应力: 95.07 MPa
计算应力: 95.07 MPa
误差: 0%
```

#### 测试4：温度应力符号
```
输入: 温度荷载 25°C
公式: σ = -αEΔT (正温差产生压应力)
计算值: -61.80 MPa
结论: ✅ 符号正确
```

#### 测试5：剪应力系数
```
公式: τmax = 2.0 × τavg (圆管)
计算比值: τmax / τavg = 2.0
结论: ✅ 系数正确
```

### 3.3 挠度计算测试

#### 测试6：竖向挠度
```
输入: 标准荷载 11.287 kN/m (自重+防腐+附加+水重)
跨长: 20000mm
惯性矩: 4.05×10⁹ mm⁴
弹性模量: 206000 MPa

公式: f = 5qL⁴/(384EI)
理论值: 28.21 mm

```

### 3.4 稳定计算测试

#### 测试7：临界压力

#### 测试8：真空压强计算
